# Smart Expense Tracker (Java)

## Problem

Students often fail to track and manage daily expenses.

## Solution

A Java-based console application that helps users record, analyze, and control their spending.

## Unique Features

* Budget tracking with warnings
* Category-wise analysis with graphs
* Search and delete functionality
* Custom date input

## Technologies

* Java
* OOP
* File Handling
* Collections
* LocalDate API

## How to Run

javac *.java
java Main
